PROJECT DETAILS:
Name : Market potential estimation

Project structure and description:

    1. demandforecastingusingHWandArima.py : implemenation of Auto arima and Holt winter's method to predict forecast for a market. This script runs both the method
    on the market and prompts the user with the best model out of the two to choose

    2.finding_model.py: This file contains the code for other models that were examnined to solve the problem along with the rmse value for each of the models, the
    project documentation explains in details, why these models were rejected

    3. gen_html : contains generated html with  graphs for all the models along with the rmse value for the analysis per market

    4. data: contains the data files used by the project, unzipped and combined
